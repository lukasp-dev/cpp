#### Manchester encoding
- 데이터 비트(0, 1)를 전송하기 위해 **전압 레벨의 변화**를 이용하는 디지털 라인 코딩 방식.
- 데이터 정송과 클럭 동기화를 하나의 신로오 동시에 해결.
- 초기 이더넷(10BASE-T 이전의 10BASE5, 10BASE2) 통신 표준에 사용됨.

#### What's in a Home Router
![image](https://ars.els-cdn.com/content/image/3-s2.0-B9781932266825500076-f01-06-9781932266825.jpg)
- ***External Network:*** 인터넷 (즉, WAN)
- ***Router:*** 외부 네트워크(WAN)와 내부 네트워크(LAN)를 구분하고, 트래픽을 라우팅함.
- ***Switch:*** 라우터의 LAN 포트(Port 1~5 중 일부)에 연결되어, 여러 컴퓨터(Computer A 등)와 네트워크 분석기(Network Analyzer)를 같은 내부망에 묶어줌. 즉, 인터넷 -> 라우터 -> 스위치 -> 여러 기기 순으로 연결돼 있고, 스위치는 내부 LAN의 허브 역할

#### Performance metrics
***Transmission time/end-to-end latency:*** $S + T_w + T_f + R$

***Message throughput*** = message-size/end-to-end-latency

- $S$: processing overhead at sender
- $R$: processing overhead at receiver
- $T_w$: how long it takes to put the message on the network?
    - $T_w =$ message-size/network-bandwidth
    - Ex: 21MB at 56kb/sec
    - (21 * 2^20 * 8)/56000
    - 3146 seconds
- $T_f$
    - the speed of light in the transmission medium (~0.5c to 1c) determines $T_f$.
    - How far is it from P1 to P2? 
    - Queueing delay is an additional term.
